# アプリ名
MY_PAGE

# 概要
- 1対多のコミュニケーションアプリ。
- ブログと画像投稿がメインの機能になります。
- 詳細

    - 管理ユーザー email: "admin@example.jp",password: "11111111"
    - 一般ユーザー email: "test@example.jp",password: "11111111",

# 作成背景
- 

## 機能
- ユーザー登録時にadminを選択すると管理者でログインができる/しなければ一般ユーザー。
- 管理者はブログや写真を投稿できる。
- 画像の投稿は１枚、ブログは１記事から。
- 一般ユーザーはブログに対してコメントが可能（3月2日エラー発生のため削除）
- 一般ユーザーは閲覧・コメントが可能。
